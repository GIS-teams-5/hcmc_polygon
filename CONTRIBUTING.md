# Hướng dẫn đóng góp

Chúng tôi rất hoan nghênh các đóng góp! Vui lòng làm theo các bước dưới đây khi muốn đóng góp mã.

## Quy trình đóng góp

1. Fork repository này.
2. Tạo nhánh mới cho thay đổi của bạn.
3. Gửi pull request (PR) để được xem xét.

## Quy tắc

- Tất cả các pull request phải đi kèm với mô tả chi tiết.
- Vui lòng không gửi pull request nếu không có kiểm tra hoặc tài liệu đầy đủ.
